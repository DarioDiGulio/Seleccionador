Seleccionador

El seleccionador es una interfáz que va a ayudar a tomar decisiones complejas de una forma sencilla. Lo único que tenés que hacer es seguir las instrucciones de cada paso y verás el resultado.

Paso 1

Opciones

Ingresá las opciones que estás evaluando.

Paso 2

Variables

Ingresá las variables que deberías tener en cuenta para tomar esta decisión.

Paso 3

Prioridades

Ordená las variables según la prioridad que creas que deben tener.

Variables

Paso 4

Valores

Ordená los valores internos de menor mayor, donde 1 es el peor valor. Hacé esto con cada variable comparando las opciones entre sí.

Variables
